## Swift 6 Priority Checks

### 1. Concurrency Safety
```text
□ @MainActor for UI updates
□ Sendable conformance for shared types
□ Actor isolation for shared state
□ Task cancellation handling
```

### 2. SwiftUI Best Practices
```text
□ Small, focused views (<100 lines)
□ Extract subviews for reusability
□ @Observable for state (not ObservableObject)
□ Preview providers for all views
```

### 3. Protocol-Oriented Design
```text
□ Protocols in dedicated files
□ Protocol extensions for defaults
□ Dependency injection via protocols
□ Mock-friendly architecture
```

### 4. Memory Management
```text
□ [weak self] in closures
□ No retain cycles
□ Proper Task cancellation
□ @State/@StateObject ownership
```

## Example: ViewModel Review

```swift
// Before elicitation
class UserViewModel: ObservableObject {
    @Published var users: [User] = []

    func load() {
        Task {
            users = try await api.fetchUsers()
        }
    }
}

// After CONC-01 + TEST-01 + ARCH-01
@MainActor
@Observable
final class UserViewModel {
    private(set) var users: [User] = []
    private(set) var isLoading = false
    private(set) var error: Error?

    private let userService: UserServiceProtocol

    init(userService: UserServiceProtocol = UserService()) {
        self.userService = userService
    }

    func load() async {
        isLoading = true
        error = nil

        do {
            users = try await userService.fetchUsers()
        } catch {
            self.error = error
        }

        isLoading = false
    }
}
```
