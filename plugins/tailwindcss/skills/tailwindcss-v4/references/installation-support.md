# Tailwind CSS v4 — Installation & Browser Support

## Installation

```bash
npm install -D tailwindcss @tailwindcss/postcss
# or for Vite
npm install -D @tailwindcss/vite
# or for CLI
npm install -D @tailwindcss/cli
```

## Browser Support

- Safari 16.4+
- Chrome 111+
- Firefox 128+
