# CHANGELOG Entry Templates

Load when writing the actual CHANGELOG entry.

## Standard repository

```markdown
## [X.Y.Z] - DD-MM-YYYY

- commit message from Step 1
```

## Plugin marketplace

```markdown
## [X.Y.Z] - DD-MM-YYYY

- type(plugin-name): description (plugin-name X.Y.Z)
```

Every numeric or behavioral claim must be proved by a command before it is written.
