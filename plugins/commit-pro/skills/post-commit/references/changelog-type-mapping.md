# CHANGELOG Type Mapping

Load when: writing a CHANGELOG entry (Step S1 or Step M4) and you need the commit-type → CHANGELOG-prefix mapping.

| Commit Type | CHANGELOG Prefix |
|-------------|-----------------|
| `feat` | Added |
| `fix` | Fixed |
| `refactor` | Changed |
| `docs` | Documentation |
| `perf` | Performance |
| `test` | Tests |
| `chore` | Maintenance |
| `style` | Style |
| `ci` | CI/CD |
| `build` | Build |
